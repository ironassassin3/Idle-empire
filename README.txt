IDLE EMPIRE — Criminal Idle Game
=================================

SETUP
-----
1. Make sure Python 3.10 or newer is installed.
   Download at: https://www.python.org/downloads/
   (Check "Add Python to PATH" during install on Windows.)

2. Open a terminal in this folder.
   On Windows: hold Shift, right-click inside the folder, choose
   "Open PowerShell window here" or "Open Terminal here".

3. Install the required library:

   pip install -r requirements.txt

4. Run the game:

   python main.py


QUICK START (Windows double-click)
-----------------------------------
Double-click run.bat to launch the game directly.


CONTROLS
--------
- Mouse: click everything
- ESC: pause / back
- All other controls are shown in-game.


CRASH REPORTS
-------------
If the game crashes, a file called crash.log will appear in this folder.
Please send that file when reporting a bug.


SYSTEM REQUIREMENTS
--------------------
- Python 3.10+
- Windows / macOS / Linux
- No dedicated GPU required
